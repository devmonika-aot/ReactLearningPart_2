---use of useEffect--
As we know reducer method should be pure method, which means It should not contain any SIDE-EFFECT and also
shouldn't have any async method.

So if we have to execute any http call or async call inside recuder method, inorder to do that 
we can use use effect in the component which is dependent upon reducer mthod i.e wherever we are using useSelector()

and useEffect can be dependent upon value return vy useSelector()
-----------eg below----------
